# Med-Research-Skill & Deep Research Knowledge Base

## 1. Overview
本專案為高階醫學研究與自動化深度挖掘知識庫，專注於「胎心音分析 (CTG)」與「輻射防護 (Lead Apron)」。

## 2. Standard Research Loop (v7.8 Master)
1. **Scope**: 定義研究主題，動態調整分析矩陣行數 (3-7 rows)。
2. **Retrieve**: 執行 Dual-Link Protocol (`[Local](<../papers/filename.pdf>) | [Web](URL)`).
3. **Synthesize**: 產出 10 欄位高密度技術分析矩陣。
4. **Deep Dive**: 執行數學建模分析與數據工程瓶頸剖析。

## 3. Trusted Medical Research Path
為確保研究具備高信度，建議優先使用以下工具路徑：
- **Primary Database**: PubMed (via MCP Server) / ClinicalTrials.gov
- **AI-Powered Synthesis**: Elicit / Consensus.ai (用於證據綜合與 JCR 級別篩選)
- **Technical Verification**: Google Scholar / IEEE Xplore / Nature Medicine

## 4. JCR/Impact Factor Reference
研究分析建議優先對標以下高影響力期刊：
- **General Medicine**: NEJM (IF: 158), Lancet (IF: 145), JAMA (IF: 120), BMJ (IF: 105)
- **Radiology/Imaging**: Radiology (IF: 19.7), IEEE Trans Med Imaging (IF: 10.6), European Radiology (IF: 5.9)
- **Digital Health/AI**: Lancet Digital Health (IF: 30.8), Nature Digital Medicine (IF: 15.2)

## 5. Technical Maintenance
- **Data Engineering**: 需確保數據標準化 (Normalization) 與標註一致性 (Consensus Protocol)。
- **Repo Structure**: 嚴格遵守 v7.8 Master 規範，所有報告存於 `reports/`，實體文獻存於 `papers/`。
